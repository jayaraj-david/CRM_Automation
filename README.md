# AI Interview Lead Generator

**Automated lead finder for Indian AI & SaaS startups (1-200 employees)**

Finds promising startups on Wellfound, hunts for founders/CTOs on LinkedIn, validates them with LLM, and adds high-quality leads directly to Google Sheets.

---

## Features

- Smart discovery from Wellfound using multiple targeted queries
- Automatic employee size filtering (`1-200` employees by default)
- Precise LinkedIn decision-maker search (Founder, CEO, CTO, VP Engineering, etc.)
- Strict LLM validation to avoid past employees or fake matches
- **No-contact smart skip** — companies without a decision-maker are logged in a separate `NoContact` tab and **never tried again**
- Personalized "Why Good Fit" reason generated by Groq LLM
- Clean, clutter-free Google Sheets output
- Retry logic for flaky searches

## Quick Setup

### 1. Clone & Install

```bash
git clone https://github.com/yourusername/ai-interview-lead-generator.git
cd ai-interview-lead-generator
pip install -r requirements.txt
```

### 2. Create config.py

```python
# config.py
GROQ_API_KEY = "gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

GOOGLE_CREDENTIALS_FILE = "credentials.json"
SPREADSHEET_ID = "your_google_spreadsheet_id_here"
CRM_SHEET_NAME = "Leads"

MAX_COMPANIES = 15
MAX_EMPLOYEES = 200
STRICT_SIZE_FILTER = True
LEAD_QUERY = ""   # Optional custom discovery query
```

### 3. Google Sheet Setup

1. Create a Google Sheet
2. Copy the Spreadsheet ID from the URL
3. Create two tabs:
   - `Leads` (main CRM)
   - `NoContact` (rejected companies — script will create header automatically)
4. Download your Google Service Account JSON → rename to `credentials.json`

### 4. Run

```bash
python main.py
```

## How It Works

1. **Discovery** → Searches Wellfound → LLM extracts real company names
2. **Deduplication** → Skips companies already in Leads or NoContact
3. **Size Check** → Filters companies by employee count
4. **Contact Hunt** → Smart LinkedIn search + LLM validation

**Result:**
- ✅ **Decision-maker found** → Added to Leads tab
- ❌ **No decision-maker** → Logged in NoContact tab (permanently skipped)

## Troubleshooting

- **Lots of "SEARCH DEAD" messages** → DDGS is rate-limited. Reduce `MAX_COMPANIES` or wait a few minutes.
- **No companies found** → Add your own query to `LEAD_QUERY` in config.py
- **Sheets permission error** → Make sure `credentials.json` has Editor access to the spreadsheet.
